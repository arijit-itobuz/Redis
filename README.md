# Redis

---

## About

- In memory data-store

---

## Configuration

- redis server port: `6379`
- redis insights port: `8001`

---

## Set-Up

- `docker compose -f ./compose.db.yaml up --build -d`
- `docker container exec -it redis_stack sh`
  - `redis-cli`
